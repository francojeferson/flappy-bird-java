# Product

## Purpose

A simple Flappy Bird game clone built in Java. The goal is to create an entertaining arcade-style game where players
control a bird navigating through obstacles.

## Problems It Solves

- Provides a fun, casual gaming experience
- Serves as a learning project for Java game development
- Demonstrates 2D game mechanics: physics, collision detection, scoring

## How It Should Work

The player controls a bird that continuously moves forward. Tapping or pressing a key makes the bird flap upward.
Gravity pulls the bird down. The bird must navigate through gaps between pipes without colliding. Score increases for
each pipe successfully passed. Game ends on collision with pipes or ground.

## User Experience Goals

- Simple one-button control (spacebar or mouse click)
- Smooth animations and responsive controls
- Clear visual feedback for scoring and game over
- Easy restart after game over
